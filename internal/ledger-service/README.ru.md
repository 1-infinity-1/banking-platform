# Ledger Service

> [English](README.md) · [Русский](README.ru.md)

Ledger Service отвечает за бухгалтерский и аудиторский учет операций.  
Сервис асинхронно получает события о завершенных транзакциях, сохраняет неизменяемые записи в журнал проводок и формирует выписки по счетам за период.

Основные задачи:
- потребление событий о завершенных транзакциях;
- хранение immutable log;
- аудит операций;
- построение выписок по счетам;
- отделение учетной модели от transactional модели.

## gRPC API (запланировано)

| Метод | Запрос | Ответ | Описание |
|-------|--------|-------|----------|
| `GetStatement` | `GetStatementRequest` (account_id, from, to) | `Statement` | Выписка по счёту за указанный период |
