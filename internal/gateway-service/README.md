# Gateway Service

> [English](README.md) · [Русский](README.ru.md)

Gateway Service is the single entry point for client applications.  
It exposes a REST API for web and mobile clients, routes requests to internal microservices, validates JWT tokens, and adapts responses to client-facing formats.

Main responsibilities:
- REST API for clients (generated by ogen from `api/openapi.yaml`);
- JWT Bearer authentication via `SecurityHandler`;
- authentication and session management via Auth Service;
- account management via Account Service;
- fund transfers and replenishments via Transaction Service;
- account statement retrieval via Ledger Service.

---

## REST API

Default port: **8081**

REST code is generated by [ogen](https://github.com/ogen-go/ogen) from `api/openapi.yaml`.  
To regenerate after editing the spec: `make generate` inside `internal/gateway-service/`.

Protected endpoints require `Authorization: Bearer <jwt>` header.

| Method | Path | Auth | Upstream |
|--------|------|------|----------|
| `GET` | `/ping` | — | — |
| `POST` | `/api/v1/auth/register` | — | auth-service |
| `POST` | `/api/v1/auth/login` | — | auth-service |
| `POST` | `/api/v1/auth/logout` | JWT | auth-service |
| `POST` | `/api/v1/auth/refresh` | — | auth-service |
| `POST` | `/api/v1/accounts` | JWT | account-service |
| `GET` | `/api/v1/users/{user_id}/accounts` | JWT | account-service |
| `GET` | `/api/v1/accounts/{account_id}` | JWT | account-service |
| `GET` | `/api/v1/accounts/{account_id}/balance` | JWT | account-service |
| `PATCH` | `/api/v1/accounts/{account_id}/status` | JWT | account-service |
| `POST` | `/api/v1/transactions/transfer` | JWT | transaction-service |
| `POST` | `/api/v1/transactions/replenish` | JWT | transaction-service |
| `GET` | `/api/v1/accounts/{account_id}/transactions` | JWT | transaction-service |
| `GET` | `/api/v1/transactions/{transaction_id}` | JWT | transaction-service |
| `GET` | `/api/v1/accounts/{account_id}/statement` | JWT | ledger-service |

## gRPC Clients

| Service | Env prefix | Default address |
|---------|------------|-----------------|
| auth-service | `GATEWAY_AUTH_GRPC_*` | `localhost:8082` |
| account-service | `GATEWAY_ACCOUNT_GRPC_*` | `localhost:8083` |
| transaction-service | `GATEWAY_TRANSACTION_GRPC_*` | `localhost:8084` |
| ledger-service | `GATEWAY_LEDGER_GRPC_*` | `localhost:8085` |

## Configuration

Loaded from `local.env` (local run) or `docker.env` (Docker). Env prefix: `GATEWAY`.

| Variable | Default | Description |
|----------|---------|-------------|
| `GATEWAY_LOG_LEVEL` | `info` | Log level |
| `GATEWAY_HTTP_PORT` | `8081` | HTTP listen port |
| `GATEWAY_AUTH_GRPC_HOST` | `localhost` | auth-service gRPC host |
| `GATEWAY_AUTH_GRPC_PORT` | `8082` | auth-service gRPC port |
| `GATEWAY_ACCOUNT_GRPC_HOST` | `localhost` | account-service gRPC host |
| `GATEWAY_ACCOUNT_GRPC_PORT` | `8083` | account-service gRPC port |
| `GATEWAY_TRANSACTION_GRPC_HOST` | `localhost` | transaction-service gRPC host |
| `GATEWAY_TRANSACTION_GRPC_PORT` | `8084` | transaction-service gRPC port |
| `GATEWAY_LEDGER_GRPC_HOST` | `localhost` | ledger-service gRPC host |
| `GATEWAY_LEDGER_GRPC_PORT` | `8085` | ledger-service gRPC port |
| `GATEWAY_JWT_SECRET` | **required** | HS256 secret for JWT validation |

## Project Structure

```text
internal/gateway-service/
├── api/
│   ├── openapi.yaml            # OpenAPI 3.0 spec (source of truth for REST)
│   └── ogen/                   # generated by ogen — do not edit manually
├── internal/
│   ├── app/
│   │   ├── application.go      # composition root: wires all clients, services, transport
│   │   └── srv/                # HTTP server wrapper
│   ├── clients/
│   │   ├── auth/               # gRPC client → auth-service
│   │   ├── account/            # gRPC client → account-service
│   │   ├── transaction/        # gRPC client → transaction-service
│   │   └── ledger/             # gRPC client → ledger-service
│   ├── config/                 # envconfig struct (prefix GATEWAY)
│   ├── models/                 # domain types and typed errors
│   ├── services/
│   │   ├── auth/               # auth business logic wrapper
│   │   ├── account/            # account business logic wrapper
│   │   ├── transaction/        # transaction business logic wrapper
│   │   ├── ledger/             # ledger business logic wrapper
│   │   └── management/         # user management (CreateUser)
│   └── transport/
│       ├── handlers.go         # GatewayHandler struct and service interfaces
│       ├── middleware/
│       │   └── auth.go         # JWTSecurityHandler (ogen SecurityHandler)
│       ├── error.go            # NewError: domain errors → HTTP responses
│       └── *.go                # one file per endpoint
├── Makefile
└── main.go
```
