# Gateway Service

> [English](README.md) · [Русский](README.ru.md)

Gateway Service is the single entry point for client applications.  
It exposes a REST API for web and mobile clients, routes requests to internal microservices, aggregates data, validates authorization, and adapts responses to client-facing formats.

Main responsibilities:
- REST API for clients;
- authentication via Auth Service;
- aggregation of user and account data;
- request routing to Account, Transaction, and Ledger services;
- unified external API boundary of the platform.

---

## REST API

Default port: **8081**

REST code is generated by [ogen](https://github.com/ogen-go/ogen) from `api/openapi.yaml`.  
To regenerate after editing the spec: see `Makefile`.

| Method | Path | Status | Upstream |
|--------|------|--------|----------|
| `GET` | `/ping` | implemented | — |
| `POST` | `/api/v1/auth/login` | planned | auth-service |
| `POST` | `/api/v1/auth/logout` | planned | auth-service |
| `GET` | `/api/v1/users/me` | planned | auth-service + account-service (aggregated) |
| `POST` | `/api/v1/accounts` | planned | account-service |
| `GET` | `/api/v1/accounts` | planned | account-service |
| `GET` | `/api/v1/accounts/{id}/balance` | planned | account-service |
| `POST` | `/api/v1/transactions/transfer` | planned | transaction-service |
| `POST` | `/api/v1/transactions/replenish` | planned | transaction-service |
| `GET` | `/api/v1/transactions/history` | planned | transaction-service |
| `GET` | `/api/v1/statements/{accountId}` | planned | ledger-service |

## Project Structure

```text

├── api/
│   ├── openapi.yaml            # OpenAPI 3.0 spec (source of truth for REST)
│   └── ogen/                   # generated by ogen — do not edit manually
│       ├── oas_handlers_gen.go
│       ├── oas_router_gen.go
│       ├── oas_schemas_gen.go
│       └── ...
├── internal/
│   └── transport/
│       ├── handlers.go         # handler implementations
│       ├── ping.go             # GET /ping
│       └── error.go            # error response helpers
```
